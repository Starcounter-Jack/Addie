## Building and running

You need a C/C++ compiler and CMake. 

### Linux

```bash
$ mkdir ~/addie
$ cd ~/addie
$ sudo apt-get install cmake
$ git clone https://github.com/Starcounter/Addie.git ./
$ cmake ./
$ make
$ ./addie
```
